Welcome to BlueOS 7's documentation!
====================================

**BlueOS 7** is the 7th generation of Blue OS that began development in April 3rd of 2022. 
This new OS is planned with many exciting features that aims to push the limits of Scratch.
This documentation will help developers of BlueOS get started *faster* and understand *better* of 
BlueOS infrastructure.

Any questions please reach out to us in our `Discord server <https://discord.gg/YqdXfnXJem>`_ or 
place a comment on our `Scratch studio <https://scratch.mit.edu/studios/31229287/comments>`_

.. note::

   This project is under active development. Documentation may be incomplete or subject to changes.
